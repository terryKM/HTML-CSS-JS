Les balises DIV

La balise DIV, est une balise neutre de type block. Elle me servira pour structurer ma page.

Les balises block agissent sur un Axe verticale et donc génére un retour à la ligne.
Je pourrais toutefois changer leurs comportements avec du CSS.

Les balises Inline elles, agissent sur un axe Horizontale et ne génère aucun retour à la ligne.
Je pourrais toutefois changer leurs comportement avec du CSS.

Les formulaires :

  • La Balise FORM permet de créer le formulaire
  • name : Permet de donner un nom au formulaire
  • action : Cet Attribut est OBLIGATOIRE et permet de renseigner le script / la page vers laquelle l'utilisateur sera redirigé lors de l'envoi du Formulaire. Cette page va recevoir l'ensemble des données saisie par mon utilisateur.
  • method : Permet de préciser quelle méthode [GET | POST] doit être utilisé par le navigateur pour transmettre les données saisies par l'utilisateur vers la page ACTION.














Créer un lien en HTML

  • La balise "a" permet de créer un lien vers une autre ressource.(Images, Vidéos, PDF, ZIP,...)

  • 'Attribut href: Permet de définir le chemin vers le fichier à lier.

  • l'Attribut target: Prend plusieur valeurs : "_blank pour ouvrir le lien dans une nouvelle fenêtre par exemple

  • l'Attribut title : Permet de donner un titre au lien.Ce titre sera afficher dans une petite bulle au survol du lien par la souris
